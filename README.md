# Introduction to Machine Learning with Python

This repository contains code reproduction and theoretical summaries
from the book "Introduction to Machine Learning with Python"
by Andreas C. Müller and Sarah Guido.

## Contents

Chapter 1 - Introduction
Overview of machine learning concepts including supervised and
unsupervised learning.

Chapter 2 - Supervised Learning
Implementation and explanation of various supervised learning
algorithms such as KNN, linear regression, decision tree,
random forest, and SVM.

Chapter 3 - Unsupervised Learning
Clustering and dimensionality reduction techniques including
K-Means, PCA, and DBSCAN.

Chapter 4 - Data Representation
Feature engineering techniques and preprocessing methods.

Chapter 5 - Model Evaluation
Evaluation techniques including cross-validation and grid search.

Chapter 6 - Algorithm Chains and Pipelines
Combining preprocessing and machine learning models using pipelines.

Chapter 7 - Working with Text Data
Text representation methods such as Bag of Words and TF-IDF.
